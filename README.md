## Features
* Draw a track
* Adjust marker spacing
* Adjust gate spacing
* Generate a track
* Generate race
* Single line marker (in the middle of track)
* Double line marker (width of your choice)
* Different maps (*Liftoff Arena* and *The Drawing Board*)
* Different marker types
